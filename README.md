# Oregon Wildfire Cause Classification

A machine learning pipeline classifying the causes of Oregon wildfires using ensemble methods. Built as a collaborative DATA 505 group project at Willamette University.

## Models

- Random Forest
- Gradient Boosting
- K-Nearest Neighbors
- Ensemble (weighted)

## Results

- Best individual model: Random Forest (~79% accuracy)
- Ensemble CV accuracy: ~78%
- Gradient Boosting ROC-AUC: 0.7939

## Project Structure

- `project/` — final combined notebook
- `presentation/` — slide deck

- **Language:** Python
